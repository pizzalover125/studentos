minimalist organization tool for students to manage their academics

### inspiration
im a student and no tool i use is designed by students. all of it is bloated and unappealing. there isn't really a central hub either. i wanted to change that.

### features
- manage classes
- manage homework
- manage upcoming tests
- pomodoro timer
- manage volunteer hours
- manage extracurcciulars you are involved in
- music w/ soundcloud
- highly customizable (font, color scheme, and more)!
